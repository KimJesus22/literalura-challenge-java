Acerca de Literalura

Literalura es un desafío de Alura Latam que te permite interactuar con la API de Gutendex para crear tu propio catálogo de libros. Este proyecto demuestra habilidades en el desarrollo de aplicaciones web con Spring Boot y la creación de una interfaz de usuario interactiva.
